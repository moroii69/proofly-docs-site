# Diabetes Management

## Overview
The diabetes management module provides comprehensive analysis of diabetes-related health metrics.

```python py filename="diabetes.py"  copy
# Diabetes analysis example
from proofly import HealthAnalyzer
from proofly.models import DiabetesMetrics

analyzer = HealthAnalyzer()
metrics = DiabetesMetrics(
    blood_glucose=120,
    hba1c=6.5,
    blood_pressure=130
)
```

## Metrics
- Blood Glucose (mg/dL)
- HbA1c (%)
- Blood Pressure (mmHg)
