COPD

## Overview
The COPD management module provides comprehensive analysis of COPD-related health metrics.

```python py filename="copd.py"  copy
# COPD analysis example
metrics = COPDMetrics(
    oxygen_saturation=95,
    peak_flow=350,
    respiratory_rate=18
)
```

## Metrics
- Blood Glucose (mg/dL)
- HbA1c (%)
- Blood Pressure (mmHg)
